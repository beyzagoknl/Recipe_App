## 5. Must Haves

- A user should be able to search a meal name and see related results.
- A user should be able to click on a button and a pop-up page opens.
- A user should be able to see the details such as ingredients and recipes belonging to the selected meal.
- A user should be able to go to the home page to make a new search.

## 6. Nice-to Have

- A user can use a surprise me button and see a random meal.
- A user can click on a link and watch a related YouTube video.
